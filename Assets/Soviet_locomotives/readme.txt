================TEM-2 (Soviet Locomotives)===============

Requires Unity 4.3.4 or higher.

This package includes model of Soviet locomotive TEM-2 with 3 maps in 2 texture files:

- 2048x2048 - diffuse (RGB) + gloss (Alpha) in PSD format;

- 2048x2048 - normal (RGB) in PSD format.


Demo scene also included.
tris count of mesh: 4542 tris

model location: \Assets\Soviet_locomotives\TEM-2 model\TEM_2.fbx
demo scene location: \Assets\Soviet_locomotives\demo scene loco\demo_locomotive.unity
